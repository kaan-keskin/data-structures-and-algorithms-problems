# Last Digit of the Sum of Fibonacci Numbers

## Problem Introduction

The goal in this problem is to find the last digit of a sum of the first 𝑛 Fibonacci numbers.

## Problem Description

**Task:** Given an integer 𝑛, find the last digit of the sum 
𝐹(0) + 𝐹(1) + · · · + 𝐹(𝑛).

**Input Format:** The input consists of a single integer 𝑛.

**Constraints:** 0 ≤ 𝑛 ≤ 10^14

**Output Format:** Output the last digit of 𝐹(0) + 𝐹(1) + · · · + 𝐹(𝑛).
